# Changelog

## [1.0.0] — 2026-05-26

- Initial release: `init` and `update` commands.
- Auto-detects fresh folders vs existing projects.
- Installs both Claude Code and Codex configurations by default; `--only=claude` / `--only=codex` flags available.
- Bundled snapshot of Virgilio skills, hooks, and templates — no network required at install time.
- Cross-platform: macOS, Linux, Windows. Node.js 18+.
- User guide in English (primary), Italian, German.
